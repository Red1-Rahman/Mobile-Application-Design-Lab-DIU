# Dart Programming Examples

This repository contains comprehensive Dart programs covering various functionalities and syntax. Each program is in a separate file for easy learning and reference.

## Programs Overview

### Basic Concepts
1. **01_hello_world.dart** - Basic Hello World program
2. **02_variables_datatypes.dart** - Variables and data types (int, double, String, bool, dynamic, var, const, final)
3. **03_operators.dart** - All operators (arithmetic, relational, logical, assignment, ternary, null-aware, cascade)
4. **04_control_flow.dart** - Control flow statements (if-else, switch-case, for, while, do-while, break, continue)
5. **05_functions.dart** - Functions (simple, parameters, return values, arrow functions, optional, named, anonymous, recursive)

### Data Structures
6. **06_strings.dart** - String operations and methods
7. **07_lists.dart** - Lists/Arrays (creation, modification, iteration, transformations)
8. **08_sets.dart** - Sets (unique collections, set operations)
9. **09_maps.dart** - Maps/Dictionaries (key-value pairs)

### Object-Oriented Programming
10. **10_classes_objects.dart** - Classes and objects (constructors, methods, getters, setters, static members, factory)
11. **11_inheritance.dart** - Inheritance (extends, override, abstract classes, polymorphism)
12. **12_interfaces_mixins.dart** - Interfaces and mixins (multiple inheritance, code reuse)
13. **13_enums.dart** - Enumerations (basic and enhanced enums)

### Advanced Features
14. **14_exception_handling.dart** - Exception handling (try-catch-finally, custom exceptions)
15. **15_generics.dart** - Generics (generic classes, methods, constraints)
16. **16_async_programming.dart** - Asynchronous programming (Future, async/await, Stream)
17. **17_file_operations.dart** - File I/O operations
18. **18_null_safety.dart** - Null safety features
19. **19_typedef_callable_classes.dart** - Type aliases and callable classes
20. **20_extension_methods.dart** - Extension methods

### Problem Solving Programs
21. **21_fibonacci_series.dart** - Fibonacci sequence (iterative, recursive, memoization)
22. **22_palindrome_check.dart** - Palindrome checking (string, number, list)
23. **23_string_reversal.dart** - String reversal techniques (multiple methods)
24. **24_prime_numbers.dart** - Prime number operations (checking, generation, factors)
25. **25_factorial_calculation.dart** - Factorial calculation (iterative, recursive, BigInt)
26. **26_array_problems.dart** - Common array problems (largest, reverse, sum, duplicates, etc.)

## Running the Programs

To run any program, use the Dart command:

```bash
dart run <filename>.dart
```

For example:
```bash
dart run 01_hello_world.dart
```

## Prerequisites

- Dart SDK installed (version 2.12 or higher for null safety features)
